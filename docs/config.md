
#  **data config** command


## Options:
```
-h, --help               Output usage information
```
    
## Usage:
```
$ data config
$ data configure
```
  
## Example:
```
# Sets Up Configurations for DataHub 
# config file is saved in ~/.datahub/config 
$ data config
```
