
#  Set configurations

## Usage:

```
$ data config[ure]
```

## Options:

```
-h, --help               Outputs usage information
```

## Example:

```
# Set up configurations for DataHub
# The config file is saved in ~/.datahub/config
$ data config

> Username: DataGeek
> Your secret token (input hidden): Y0uR53cr3tt0KeN
> Server URL: https://datapackaged.com/
```
