
#  **data info** command


## Options:
```
-h, --help               Output usage information
```
    
## Usage:
```
# Get Data Package info for given publisher and package:
$ data info <publisher>/<package>
```
  
## Example:
```
# Get Data Package info for given core publisher and co2-ppm package: 
$ data info core/co2-ppm
```
